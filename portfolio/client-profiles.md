# Organization Footprint and Client Profile Layer

This page is a public working inventory. It is not a sales list and it does not
claim that every named organization is a client. The purpose is to show the
institutional range behind the portfolio while protecting private client names,
unfinished conversations, and confidential proposal work.

## How To Read This Page

| Layer | Meaning | Public rule |
| --- | --- | --- |
| Named public organization | A public institution, university, program, ecosystem, or partner that can be named without exposing a confidential client. | Name it only when the relationship or route is documented. |

## Removed From Public Claims Until Verified

The previous version mixed too many weak leads into the public footprint. These
should not be presented as clients or counterparties until there is source
evidence:

- generic opportunity radar entries where no application, meeting, or direct
  engagement is documented;
- directory-only research-center names;
- raw university/contact directories;
- unsent or exploratory email targets;
- program names that describe a funding opportunity but not a relationship;
- entities where the exact legal name, current status, or contact history is
  still unclear.

Examples that should stay in private notes unless verified: raw ANUIES/CUPIA
directory entries, uncontacted research centers, broad challenge lists, and
generic accelerator scouting rows.

## Editorial Rule

Before adding a new row, classify it as one of these:

1. client;
2. partner;
3. funder/program;
4. public institution;
5. city/public authority;
6. research or academic route;
7. ecosystem/contact route;
8. anonymized private client.

If the classification is unclear, it belongs in the verification backlog, not in
the public footprint.
